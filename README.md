# Twitter-Clone
```
Let’s make simple Twitter. Users can post tweets and see everyone’s them.
To make the project simple, we do not have a sign/login function.
So there are not user accounts and follow function. You can challenge after you finish the course.
```
Check out [LIVE Twitter clone DEMO](https://twitter-clone.loganringo.repl.co/) here!!

![twitter clone screenshot](https://user-images.githubusercontent.com/113215548/203841916-fb319a10-038f-4155-a191-e065973ca782.PNG)



## Tech used
```
* Html
* Css
* Bootstrap
* Javascript / JQuery
* Django
* Cloudinary (To host uploaded images on CDN)
```
## User Story
```
* Users can post tweets.
* Users can edit tweets.
* Users can delete tweets.
* Users can send a like to a tweet.
```
